# QR-Code-Generator
Simple QR code generator in python
## Instructions
You have to run this program in virtual enviroment

### Linux

sudo apt install python3.12-venv

python3 -m venv venv

#### Run next command to activate venv:

source venv/bin/activate

#### Install qrcode library:

pip install qrcode

#### Run code:

python main.py

#### Deactivate venv:

deactivate
